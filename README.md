# NetrunnerDB_Chinese_Tools
整合卡牌名称中文翻译功能与 Print and Play 简中卡图 PDF 生成功能。

用来在https://netrunnerdb.com/上翻译卡名，看牌组更方便。

并且新增了PNP按钮可以生成简中版卡图PDF
